# Frontend Service

A simple React application built with Vite to interact with the recommendation system.

## 🎨 Features

1.  **Content Feed**: Displays all available items (text, images, audio).
2.  **Personalized Recommendations**: Shows a "Recommended for You" section that updates in real-time as you interact with the app.
3.  **Interactions**: A "Like" button that triggers the User Tower update on the backend.
4.  **Uploader**: A form to add new content to the system. When you upload a file, it is immediately processed and becomes part of the recommendation pool.

## 🛠️ Tech Stack
- **React**: UI library.
- **Vite**: Ultra-fast frontend build tool.
- **Tailwind CSS**: For styling.
- **Lucide React**: For beautiful icons.
- **Axios**: For API calls.
