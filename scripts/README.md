# Scripts

Utility scripts for data generation and setup.

## generate_audio_seed.py

Generates a 1-second sine wave WAV file for initial seeding.

```bash
python3 scripts/generate_audio_seed.py
```

## download_models.sh (Legacy)

The model download process is now automated within the `inference` service. This script is kept for reference or manual use if needed.

```bash
bash scripts/download_models.sh
```
